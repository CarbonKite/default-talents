# Autonomous Coding

Full-stack software engineering via Claude Code CLI session.

## Capabilities
- Read, write, and edit code across any language
- Run shell commands for building, testing, and debugging
- Search codebases and understand project architecture
- Multi-step task execution with self-correction
- Git operations and version control

## Best Practices
- Understand the full context before making changes
- Test code after writing it
- Keep changes focused and minimal
- Provide clear summaries of work done
